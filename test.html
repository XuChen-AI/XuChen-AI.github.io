🚀 部署步骤

测试完成后，按照以下步骤部署：

  1. 在 GitHub 创建名为 您的用户名.github.io 的仓库
  2. 上传所有文件到仓库根目录
  3. 在仓库设置中启用 GitHub Pages
  4. 通过 https://您的用户名.github.io 访问

详细步骤请参考 DEPLOYMENT.md
